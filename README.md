# Декартов лист — построение графика функции

**Декартов лист** — это учебный проект, разработанный в рамках курса «Разработка программных средств».  
Приложение построено на **Windows Forms (C#)** и визуализирует алгебраическую кривую третьего порядка, заданную уравнением:

$$
y = \pm x \sqrt{\frac{l + x}{l - 3x}}, \quad l = \frac{3a}{\sqrt{2}}
$$

Пользователь может задавать коэффициент `a`, границы интервала и шаг вычислений. Программа выводит таблицу значений и строит график с двумя ветвями.

---

## 📋 Функциональные возможности

- Ввод параметров: `a`, `left`, `right`, `step`.
- Таблица значений с двумя ветвями (`Y+` и `Y-`).
- График с автоматическим масштабированием и обработкой разрывов.
- Сохранение данных в CSV-файл.
- Загрузка данных из CSV-файла.
- Обработка ошибок ввода, области определения и некорректных файлов.
- Приветственное сообщение при запуске.

---

## 🛠️ Технологии

- **Язык:** C#
- **Платформа:** .NET Framework (Windows Forms)
- **Графика:** GDI+ (`System.Drawing`)
- **Файлы:** CSV (с разделителем `,`, числа с точкой)
